Google Drive Setup Guide

One-time free setup — takes 10 minutes — your data syncs privately

1
Go to console.cloud.google.com and sign in with your Google account
2
Click "Select a project" → "New Project" → name it KiranaBook → Create
3
APIs & Services → Library → search "Google Drive API" → Enable
4
Credentials → Create Credentials → OAuth 2.0 Client ID
5
Application type: Web application. Add your site URL to "Authorized JavaScript origins"
6
Copy the Client ID (ends with .apps.googleusercontent.com)
7
In KiranaBook: Backup & Sync tab → Connect Google Drive → paste Client ID
8
Sign in with Google → Allow → Sync Now — your data is now on Google Drive!

⚠ If "access blocked" error: OAuth consent screen → External → add your email as test user
